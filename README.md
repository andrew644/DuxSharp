# DuxSharp
## A compiler for the Dux language written in C#
### A low level language with a modern type system using LLVM as a backend for optimized binaries.


Check DuxTestSrc directory for example code that is currently supported. These files are part of the end-to-end test suite.

Supported features:

- Function calls
- Recursion
- For loops
- If statements
- i32 variables
- Formatted output (printf)
